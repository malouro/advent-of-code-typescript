## Getting Started

**Installation & setup**
```sh
(nvm use || echo "NVM not found; make sure your Node version is $(cat .nvmrc)") && yarn install
```

**Running tests**
```sh
# Run all tests
yarn test
# Run watch mode; test as you code
yarn test --watch
# Run a specific day's tests
yarn test ./$DAY_NUMBER
```
